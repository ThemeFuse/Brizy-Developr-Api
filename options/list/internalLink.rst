Internal Link
=============

Internal link option type.

Config
------

+------------+-------------+-------------+------------------------------------------------------------------------------+
| **Name**   |  **Type**   | **Default** | **Description**                                                              |
+============+=============+=============+==============================================================================+
| size       | ``string``  | ``""``      | Specify the input size.                                                      |
|            |             |             ||                                                                             |
|            |             |             || There are 3 sizes:                                                          |
|            |             |             || - ``"short"``, for short input.                                             |
|            |             |             || - ``"medium"``, for normal input.                                           |
|            |             |             || - ``"large"``, for full width input.                                        |
|            |             |             || - ``"auto"``, for auto width.                                               |
+------------+-------------+-------------+------------------------------------------------------------------------------+

Minimal Usage
-------------

.. code-block:: javascript

    {
      id: "page",
      type: "internalLink"
    }

-----

      label: "Page url",

      type: "internalLink",
      config: {
        size: "large",
      },
    }
