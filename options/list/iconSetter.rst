Icon Setter
===========

Icon Setter option type.

Config
------

+------------+-------------+-------------+------------------------------------------------------------------------------+
| **Name**   |  **Type**   | **Default** | **Description**                                                              |
+============+=============+=============+==============================================================================+
| canDelete  | ``boolean`` | `false`     | Admit if usr can remove or not current active icon.                          |
+------------+-------------+-------------+------------------------------------------------------------------------------+

DefaultValue
-------------------

+---------------+-------------+---------------+-------------------------------------------------------------------------+
| **Name**      |  **Type**   | **Default**   | **Description**                                                         |
+===============+=============+===============+=========================================================================+
| value         | ``string``  | ``undefined`` | Any `string` value.                                                     |
+---------------+-------------+---------------+-------------------------------------------------------------------------+


Minimal Usage
-------------

.. code-block:: javascript

    {
      id: "icon",
      type: "iconSetter-dev",
    }

Usage
-----

.. code-block:: javascript

    {
      id: "icon",
      label: "Icon",
      type: "iconSetter-dev",
      config: {
        canDelete: true,
      },
    }
