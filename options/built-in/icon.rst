Icon
----

Choose a `FontAwesome <http://fontawesome.io/>`_ icon.

.. code-block:: php

    array(
        'type'  => 'icon',
        'value' => 'fa-smile-o',
        'attr'  => array( 'class' => 'custom-class', 'data-foo' => 'bar' ),
        'label' => __('Label', '{domain}'),
        'desc'  => __('Description', '{domain}'),
        'help'  => __('Help tip', '{domain}'),
    )
